### 디렉토리 구조
>'Ducks'

#### src
  - modules : 리듀서/액션 함수
  - assets : 이미지 파일
  - components : 뷰 컴포넌트
  - containers : 리덕스로 연결하여 컴포넌트를 띄우는 컨테이너
  - styles: 중복되는 styled-components
